# Fonction JsGetExtensionAllowed
Renvoie une valeur qui indique si un objet est extensible ou non.  
  
## Syntaxe  
  
```cpp  
STDAPI_(JsErrorCode) JsGetExtensionAllowed(  
   _In_ JsValueRef object,  
   _Out_ bool *value  
);  
```  
  
#### Paramètres  
 `object`  
 Objet à tester.  
  
 `value`  
 Si l’objet est extensible ou non.  
  
## Valeur renvoyée  
 Le code `JsNoError` en cas de réussite de l’opération, dans le cas contraire.  
  
## Remarques  
 Nécessite un contexte de script actif.  
  
## Configuration requise  
 **En-tête:** jsrt. h
