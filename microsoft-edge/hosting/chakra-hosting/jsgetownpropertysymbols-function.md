# Fonction JsGetOwnPropertySymbols
Obtient la liste de toutes les propriétés de symbole sur l’objet.  
  
## Syntaxe  
  
```cpp  
STDAPI_(JsErrorCode) JsGetOwnPropertySymbols(  
   _In_ JsValueRef object,  
   _Out_ JsValueRef *propertySymbols  
);  
```  
  
#### Paramètres  
 `object`  
 Objet à partir duquel obtenir les symboles de propriété.  
  
 `propertySymbols`  
 Tableau de symboles de propriété.  
  
## Valeur renvoyée  
 Le code `JsNoError` en cas de réussite de l’opération, dans le cas contraire.  
  
## Remarques  
 Nécessite un contexte de script actif.  
  
 Cette API est uniquement prise en charge en mode Microsoft Edge.  
  
## Configuration requise  
 **En-tête:** jsrt. h
